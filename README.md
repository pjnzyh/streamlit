# 文本分析与可视化应用

基于 Streamlit 构建的文本分析与可视化工具，支持从网页抓取文本并进行深度分析。

## 功能特性

### 📊 核心功能
- **网页文本抓取**：从指定URL获取文章内容
- **中文分词**：使用 jieba 进行中文分词处理
- **词频统计**：统计文章中词汇出现频率
- **数据可视化**：支持多种图表展示（词云图、柱状图、饼图、折线图、散点图、雷达图、热力图）

### 📝 文章附属分析
- **字数统计**：精确计算中文字符、英文字符、总字数及段落数量
- **阅读时长测算**：基于文本难度系数预估阅读时间
- **情感分析**：识别文章整体情感倾向（积极/消极/中性）
- **主题分类**：自动分类文章主题并生成关键词标签

### 🔄 智能辅助功能
- **图表智能推荐**：根据数据特征自动推荐合适的可视化图表
- **多文章对比**：支持多篇文章关键词对比和语义相似度分析

### 📜 历史记录
- 自动保存分析历史
- 支持重新分析和删除记录
- 数据持久化存储

## 快速开始

### 环境要求
- Python 3.8+
- 依赖包见 `requirements.txt`

### 安装步骤

```bash
# 克隆项目
git clone <repository-url>
cd pc3

# 安装依赖
pip install -r requirements.txt

# 运行应用
streamlit run app.py
```

### 启动命令

```bash
# 开发模式
streamlit run app.py --server.port 8501

# 指定端口
streamlit run app.py --server.port <port>
```

## 使用说明

### 1. 分析文章
1. 在左侧边栏输入文章URL
2. 点击「分析文本」按钮
3. 等待分析完成后查看结果

### 2. 自定义图表
- 选择颜色主题（经典蓝、活力橙、清新绿等）
- 选择词云形状
- 调整标题字体大小
- 选择显示的图表类型

### 3. 多文章对比
1. 分析完第一篇文章后点击「添加当前文章到对比」
2. 分析第二篇文章并添加到对比列表
3. 切换到「多文章对比」标签页查看对比结果

## API 说明

### TextAnalyzer 类

#### fetch_url_content(url)
获取指定URL的网页内容

**参数**：
- `url`: 网页地址（字符串）

**返回**：HTML内容（字符串）

**异常**：网络请求失败时抛出异常

#### clean_text(html_content)
清洗HTML内容，提取纯文本

**参数**：
- `html_content`: HTML内容（字符串）

**返回**：清洗后的纯文本（字符串）

#### analyze_text(text, min_length=2)
分析文本，进行分词和词频统计

**参数**：
- `text`: 待分析文本（字符串）
- `min_length`: 最小词长，默认为2

**返回**：词频计数器（Counter）

#### count_characters()
统计字符信息

**返回**：包含中文字符、英文字符、总字符数、段落数的字典

#### estimate_reading_time()
估算阅读时间

**返回**：包含分钟数、总词数、难度系数、难度标签的字典

#### analyze_sentiment()
分析情感倾向

**返回**：包含情感评分、情感标签、积极/消极词汇计数的字典

#### classify_topic()
分类文章主题

**返回**：包含主题列表、关键词标签、主题分数的字典

### HistoryManager 类

#### load_history()
加载历史记录

**返回**：历史记录列表

#### save_history(history)
保存历史记录

**参数**：
- `history`: 历史记录列表

#### add_record(url, analyzer)
添加分析记录

**参数**：
- `url`: 文章URL
- `analyzer`: TextAnalyzer实例

**返回**：更新后的历史记录列表

## 性能优化建议

1. **限制词云图数据量**：最多显示100个词汇
2. **低频词过滤**：默认过滤词频<2的词汇
3. **历史记录限制**：最多保存50条记录
4. **异步加载**：使用Streamlit的缓存机制提升性能
